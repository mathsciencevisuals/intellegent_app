# FeatureIntel Next.js Starter

This is a starter app structure for the SaaS concept discussed earlier:
- app router pages
- shared shell and components
- mock data
- route-level screens for auth, onboarding, dashboard, repository, reports, billing, settings, and roadmap

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Suggested next implementation steps

1. Replace mock data with typed API calls.
2. Add authentication with NextAuth or your SSO provider.
3. Create backend services for source ingestion and extraction jobs.
4. Add a real database for features, source artifacts, and approvals.
5. Add charts and tables from your preferred UI library.
